AFIS Pack Installer

A local tool for installing Minecraft Bedrock Dedicated Server add-on packs without manually copying UUIDs into world JSON files.

Download Release View on GitHub
Reads packs

Supports .mcpack, .mcaddon, .zip, and folders.

Finds UUIDs

Pulls pack UUID and version from manifest.json.

Updates worlds

Writes world_behavior_packs.json and world_resource_packs.json.

Local-first

No telemetry, no account login, no SFTP credentials stored.
