# Changelog

## v0.1.0

Initial local Option C release.

### Added

- Local Python installer
- Simple GUI mode with file pickers
- CLI mode
- `.mcpack`, `.mcaddon`, `.zip`, and folder support
- Manifest UUID/version reading
- Behavior/resource pack detection
- World JSON generation/update
- Duplicate UUID prevention
- Backup folder and install report
- Optional `texturepack-required=true` setting
