# Security Policy

AFIS Pack Installer is intended to run locally and should not collect or upload user files.

## Reporting issues

If you find a security issue, open a private report if available or contact the project owner directly.

## Security expectations

- Do not commit `.env` files or credentials
- Do not hardcode SFTP passwords
- Do not upload Minecraft worlds or packs without user permission
- Always create backups before modifying world pack JSON files
