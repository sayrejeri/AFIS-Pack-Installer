# Contributing

Thanks for helping improve AFIS Pack Installer.

## Good first improvements

- Improve error messages
- Add more manifest validation
- Improve Windows GUI layout
- Add uninstall/remove pack support
- Add installed pack list support
- Add SFTP upload support

## Pull request checklist

- Keep the tool local-first and safe
- Do not add telemetry
- Do not store server passwords or SFTP credentials in plain text
- Test with a dry run before editing real worlds
- Update README/CHANGELOG when adding features
